# ショートカットキー一覧

## タスク一覧ページ

> モーダルが開いていない・テキスト入力中でない場合に有効

| キー | 動作 |
|---|---|
| `N` | タスク追加モーダルを開く |
| `1` 〜 `9` | 対応するカテゴリの絞り込みをトグル（表示順） |
| `0` | 「カテゴリなし」の絞り込みをトグル |

## タスク追加・編集モーダル

| キー | 動作 |
|---|---|
| `Ctrl` + `Enter` | タスクを保存 |
| `Enter`（タスク名入力中） | メモ欄へ移動 / スマホは保存 |
| `Tab` | 次のフィールドへ移動（タスク名 → メモ → カテゴリ → 予定日） |
| `1` 〜 `9`（カテゴリ選択中） | 対応するカテゴリを選択（表示順） |
