# タスク一覧の絞り込み・カテゴリフィルター

## カテゴリフィルター

タスク一覧の上部（モバイル）またはサイドバー（PC）に表示されるカテゴリチップで、表示するタスクのカテゴリを絞り込めます。

### 操作方法

| 操作 | 動作 |
|---|---|
| カテゴリチップをタップ/クリック | そのカテゴリの選択をトグル（ON/OFF） |
| **全て選択** ボタン | すべてのカテゴリを選択（フィルタなし状態） |
| **全て解除** ボタン | すべてのカテゴリを解除（タスク非表示） |

- **カテゴリなし** チップはカテゴリ未設定のタスクに対応します
- 複数のカテゴリを同時に選択できます
- すべてのカテゴリが選択されている状態がデフォルト（フィルタなし）です
- すべてのカテゴリを解除すると、タスクが一覧に表示されなくなります

### キーボードショートカット

モーダルやテキスト入力が開いていない状態で有効です。

| キー | 動作 |
|---|---|
| `1` 〜 `9` | 表示順で対応するカテゴリの選択をトグル |
| `0` | 「カテゴリなし」の選択をトグル |

## 絞り込みフィルター（キーワード・ステータス・日付・お気に入り）

### PC の場合

サイドバー内の「絞り込み条件」パネルから操作します。変更はリアルタイムで反映されます。

### モバイルの場合

画面右下の絞り込みボタン（スライダーアイコン）をタップするとボトムシートが開きます。条件を設定して「**適用**」ボタンをタップすると一覧に反映されます。アクティブなフィルターの件数がボタン上にバッジ表示されます。

---

## 各フィルター条件の説明

### キーワード

タスク名とメモに対してテキスト検索を行います。

- 日本語入力（IME）が確定した後に検索が発火します
- 入力から 300ms 後に自動的に検索されます
- 右端の **×** ボタンでキーワードをクリアできます

### ステータス

| 選択肢 | 表示対象 |
|---|---|
| すべて | 全ステータスのタスク |
| 未完了 | PENDING 状態のタスク |
| 完了 | COMPLETED 状態のタスク |
| やらない | SKIPPED 状態のタスク |

### 日付

指定した日付に関連するタスクを表示します。以下のいずれかに一致するタスクが対象となります。

| マッチ理由 | 条件 |
|---|---|
| 予定日 | タスクの予定日（scheduledAt）が指定日と一致 |
| この日に完了 | タスクの完了日時が指定日と一致 |
| この日にやらない | タスクのやらない日時が指定日と一致 |
| この日に作成 | タスクの作成日が指定日と一致（予定日と同日の場合は除く） |

一致したタスクカードには、マッチ理由のバッジが表示されます。

日付ナビゲーションの操作:

| 操作 | 動作 |
|---|---|
| `◀` ボタン | 前日に移動 |
| `▶` ボタン | 翌日に移動 |
| **今日** ボタン | 今日の日付を設定 |
| **×** ボタン | 日付フィルタを解除 |

### お気に入り

チェックを入れるとお気に入り（★）マークしたタスクのみ表示されます。

---

## フィルターのクリア

- PC サイドバーの「クリア」ボタン: キーワード・ステータス・日付・お気に入りをまとめてリセット
- モバイルボトムシートの「クリア」ボタン: 同上
- カテゴリフィルターは独立しており、「全て選択」ボタンで初期状態に戻せます

---

## 補足

- 絞り込み条件はURLのクエリパラメータ（`?keyword=...&status=...`）に反映されます。URLを共有するとフィルタ状態も共有されます
- 絞り込みが1件以上有効な場合、ドラッグ＆ドロップによるタスクの並び替えは無効になります
- フィルタなし時のデフォルト表示順は `displayOrder` の降順です
