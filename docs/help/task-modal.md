# タスクの追加・編集モーダル

## タスク追加モーダル

### 開き方

| 操作 | 動作 |
|---|---|
| 画面右下の **＋** ボタン | タスク追加モーダルを開く |
| キーボード `N` キー | タスク追加モーダルを開く（モーダル・テキスト入力が閉じている場合） |

モーダルを開くと、タスク名入力欄に自動でフォーカスが当たります。

### 入力フィールド

#### タスク名（必須）

タスクの内容を入力します。入力量に合わせてテキストエリアが自動で縦に広がります。

#### メモ（任意）

タスクの詳細や補足を自由に入力できます。

#### カテゴリ（任意）

チップ形式で表示されるカテゴリから選択します。「なし」を選ぶとカテゴリなしになります。

- カテゴリフィルターで1つだけカテゴリを選択中の場合、そのカテゴリが初期値として設定されます

#### 予定日（任意）

| ボタン | 動作 |
|---|---|
| **なし** | 予定日を設定しない |
| **今日** | 今日の日付を設定 |
| **明日** | 明日の日付を設定 |
| **選択**（カレンダーアイコン） | 日付ピッカーを開いて任意の日付を選択 |

### 保存

- フッターの **追加** ボタンをクリック
- タスク名が空の場合はボタンが無効になります
- 保存後、タスク名・メモはリセットされます。カテゴリは保持されます（連続入力に便利）

---

## タスク編集ダイアログ

### 開き方

タスクカードのメニュー（`...`）から「編集」を選択します。

### 入力フィールド

追加モーダルと同じフィールド（タスク名・メモ・カテゴリ・予定日）を編集できます。

#### 追加モーダルとの違い

- 予定日はカレンダー入力（`<input type="date">`）で直接選択します。「今日」「明日」の素早き選択ボタンはありません
- 予定日が設定済みの場合、**クリア** ボタンで解除できます
- フッターに **キャンセル** ボタンがあり、変更を破棄してダイアログを閉じられます

### 保存

- フッターの **保存する** ボタンをクリック
- タスク名が空のまま保存しようとするとエラーメッセージが表示されます

---

## キーボードショートカット（共通）

モーダル・ダイアログ内で使えるショートカットです。

| キー | 動作 |
|---|---|
| `Ctrl` + `Enter` | タスクを保存 |
| `Enter`（タスク名入力中・PC） | メモ欄へフォーカス移動 |
| `Enter`（タスク名入力中・スマホ） | タスクを保存 |
| `Tab` | 次のフィールドへ移動 |
| `1` 〜 `9`（カテゴリ選択中） | 表示順で対応するカテゴリを選択 |
| `0`（カテゴリ選択中） | カテゴリを「なし」に設定 |

> `Enter` の動作はタッチデバイス（スマホ・タブレット）と PC で異なります。スマホでは即保存、PC ではメモ欄へ移動します。

---

## スマートフォンでの動作

- モーダルはキーボード表示時に画面上部に移動し、入力欄が隠れないよう調整されます
- モーダル外をタップするとキャンセルして閉じます
